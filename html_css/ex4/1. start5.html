절대 단위는 고정된 크기로, 디바이스에 상관없이 동일한 크기를 가집니다.

  1. px : 1픽셀(Pixels)을 1로 하는 단위로 디스플레이의 해상도에 따라 크기가 달라진다.
  2. cm : 센티미터
  3. mm : 센티미터의 1/10
  4. in : 1인치는 2.54cm
  5. pt : 1포인트는 1/72인치

상대 단위는 부모 요소나 다른 기준에 따라 크기가 달라집니다.

  1. % (퍼센트): 부모 요소에 대한 상대적인 백분율로 크기를 지정합니다.
  2. em : 부모 요소의 폰트 크기를 기준으로 폰트 크기(font-size)를 지정합니다. 예를 들어, 부모 폰트 크기가 16px이면 1em은 16px이 됩니다. font-size를 이외의 길이 관련 속성에서는 현재 요소의 font-size를 1em의 기준으로 합니다.
  3. rem: root 요소 (<html>)의 폰트 크기를 기준으로 크기를 지정합니다.
  4. vw (Viewport Width): 뷰포트 너비의 1%에 해당하는 크기. 뷰포트(Viewport)는 사용자가 웹페이지를 볼 수 있는 브라우저 창의 가시 영역을 의미합니다.
  5. vh (Viewport Height): 뷰포트 높이의 1%에 해당하는 크기.
